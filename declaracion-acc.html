Declaración de Accesibilidad

El Museo Allpa Urku se compromete a que su sitio web sea accesible para todas las personas, de acuerdo con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.2, nivel AA.

Estado de conformidad

Este sitio web es parcialmente conforme con WCAG 2.2 nivel AA. "Parcialmente conforme" significa que algunas partes del contenido todavía no cumplen totalmente el estándar, tal como se detalla a continuación.

Contenido no accesible

Estamos trabajando para corregir los siguientes puntos:

  • Los videos de las páginas "Sobre el museo" y "El Páramo" todavía no cuentan con subtítulos con el contenido real (el mecanismo técnico ya está implementado, pendiente la transcripción).
  • Los íconos de "Asistente por voz" y "Opciones de accesibilidad" del encabezado son aún un marcador de posición; esa funcionalidad todavía no está implementada.

Cómo se preparó esta declaración

Esta declaración se preparó a partir de una autoevaluación realizada por el equipo del museo en julio de 2026, revisando cada página frente a los criterios de éxito de WCAG 2.2.
